This repository contains the source code for GPA and CGPA calculator android app (in Kotlin).
The MainActivity.kt file contains the kotlin code for the app while activity_main.xml contains the layout details of android app.
The grade to points equivalence is given. A: 4.0, B+: 3.5, B: 3.0, C+: 2.5, C: 2.0, D: 1.5, F/XF: 0.0
